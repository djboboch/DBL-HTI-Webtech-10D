Visualization 1 & 2

This webapplication visualizes your newick file in two different ways. The first visualization is a sunburst visual, and the second visualization is a tree. The sunburst should be used to view the part-to-whole relationships while the tree should be used to view specific nodes. Visualization 1 can already be interacted with while visualization 2 is still in progress. In a near future the visualizations will be shown next to each other once they are also able to interact with each other.
